Daily Flow
  1. Allocate — drop the raw bot download into the Allocator → download the Assigned Excel.
  2. Email — bot groups Assigned by agent, sends one mail each, TL in CC.
  3. Compare (run 2+) — previous vs current assigned → email only the New-to-Assign list.
  4. Append — add the Assigned file to the Master Tracker (carry-overs auto-skipped).
  5. Reconcile (EOD) — Tracker × Completion Report → OK / Mismatch / Pending / Leakage.
  6. Dashboard — the reconciled file feeds Power BI.
